No installer — extract and run.

  1. Download and unzip the archive.
  2. Open the folder that contains ASCIICraft.exe.
  3. Keep everything in that folder — the game needs:
    • ASCIICraft.exe
    • tbb12.dll
    • OpenAL32.dll
    • the res/ folder
  4. Double-click ASCIICraft.exe to start in terminal mode (16-color by default).

Or from a terminal in that folder:

.\ASCIICraft.exe
.\ASCIICraft.exe --window
.\ASCIICraft.exe --mono
.\ASCIICraft.exe --window --mono

Nothing else needs to be installed. Required DLLs are included in the download.

If Windows SmartScreen warns you: this is an unsigned build — use “More info” → “Run anyway.”

If terminal mode looks wrong: use Windows Terminal, keep the window large, and restart the terminal after first launch if the color scheme was updated.

Worlds save on quit (Esc or closing the window). Older 1.0 saves migrate automatically.
