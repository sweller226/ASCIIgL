ASCIICraft is a block-based sandbox where the entire 3D world is drawn as live ASCII art. Explore procedurally generated terrain, break and place blocks, manage an inventory, and watch the world update — all through glyphs and terminal colors.

Built on ASCIIgL, a custom rendering engine that converts a full voxel scene into character cells with foreground/background colors matched to the scene.

What makes it different

Most “ASCII games” fake the look in a normal graphics window. ASCIICraft actually renders through an ASCII pipeline: each cell is a character, and colors are chosen to approximate the 3D image. The game builds its 16-color palette from its own block and item textures using perceptual color clustering (Oklab), so the palette fits the world instead of a fixed retro scheme. Depth-aware supersampling keeps edges cleaner when that scene is quantized down to the character grid.

Features

  • Procedural terrain with trees, grass, water, stone, and varied biomes
  • Minecraft-style block mining — hold to break, with tool speed, destroy stages, swing animation, and break particles
  • Block placement, inventory, hotbar, and item drops
  • First-person held items and tools
  • First-person movement — walk, sprint, sneak, and fly
  • Game modes: Survival, Creative, and Spectator (start in Spectator; cycle with P)
  • World saving on quit — chunks persist, and older 1.0 worlds migrate automatically
  • Sound and music
  • Optional blue-noise dithering (off by default; toggle with K)
  • Two render modes:
    • Terminal mode (default) — runs in Windows Terminal; applies a custom color scheme on launch
    • Window mode — native DirectX window with the same ASCII output

Launch options

ASCIICraft.exe                    Terminal mode, 16-color (default)
ASCIICraft.exe --window           Window mode
ASCIICraft.exe --mono             Terminal mode, monochrome
ASCIICraft.exe --window --mono    Window mode, monochrome

--color / -c forces the 16-color palette; --mono / --monochrome / -m forces monochrome. --terminal / -t and --window / -w select the display.

Requirements

  • Windows 10 or 11 (64-bit)
  • Windows Terminal for terminal mode
  • GPU with DirectX 11 support

Terminal tips

  • The game tries to install the Square Modern font from bundled resources on first run. If it fails, go into the resources folder and download it manually.
