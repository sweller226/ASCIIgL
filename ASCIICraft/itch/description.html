ASCIICraft is a block-based sandbox where the entire 3D world is drawn as live ASCII art. Explore procedurally generated terrain, break and place blocks, manage an inventory, and watch the world update — all through glyphs and terminal colors.

Built on ASCIIgL, a custom rendering engine that converts a full voxel scene into character cells with foreground/background colors matched to the scene.

What makes it different

Most “ASCII games” fake the look in a normal graphics window. ASCIICraft actually renders through an ASCII pipeline: each cell is a character, and colors are chosen to approximate the 3D image. The game builds its 16-color palette from its own block and item textures using perceptual color clustering, so the palette fits the world instead of a fixed retro scheme.

Features

  • Procedural terrain with trees, grass, water, and varied biomes
  • Block breaking and placement
  • Inventory, hotbar, and item drops
  • First-person movement — walk, sprint, sneak, and fly
  • Sound and music
  • Two render modes:
    • Terminal mode (default) — runs in Windows Terminal; applies a custom color scheme on launch
    • Window mode — native DirectX window with the same ASCII output

Launch options

ASCIICraft.exe                    Terminal mode, monochrome (default)
ASCIICraft.exe --window           Window mode
ASCIICraft.exe --color            16-color palette
ASCIICraft.exe --window --color   Full color in a window

Requirements

  • Windows 10 or 11 (64-bit)
  • Windows Terminal recommended for terminal mode
  • GPU with DirectX 11 support

Terminal tips

  • The game tries to install the Square Modern font from bundled resources on first run
  • Set line height to 1.0 in Windows Terminal
  • Aliased rendering gives a sharper retro look; ClearType is easier to read
